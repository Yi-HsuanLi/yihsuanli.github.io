Concordance / 調 和

Concordance / 調和

type | print design

design | yihsuanli / 李宜軒

date | Mar . 2015

"Concordance means the state of being similar to or consistent with each other". To get along with other people, we should open our mind and accept differences each other. This exhibition is named for “Concordance”. Through this exhibition, we want to describe the difference between various cultures. The poster for this exhibition is using two vivid colors and you can find square inside which shows the message that we want to convey completely. This two colors become one slowly, just like our life that we accept and concordant in different environment slowly. There are four topics of poster design: Friendship / Political party / Community / Love .
Concordance意思為調和。與人共處，就必須不斷調和融合，接納自己所需面對的一切。畫展以 concordance為主題， 希望將人與人之間、文化及文化之間相岐到相容的過程，用圖述說，畫面的主視覺色彩我選用了兩種鮮豔的色彩，而中間的色塊則是完整說明了”concordance“的過程，兩個顏色漸漸的調和為一個顏色，就像我們生活裡，漸漸地接受及融合在不同的環境裡。 此次海報分別設計了四款顏色代表：友誼/政黨/愛情/社會
